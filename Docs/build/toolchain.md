# GARRISON — Toolchain

M0 pins the prototype toolchain so everyone compiles against the same editor
and networking package.

- **Unity Editor:** `6000.4.9f1`
- **PurrNet:** `dev.purrnet.purrnet` `1.19.1`, installed from
  `https://github.com/PurrNet/PurrNet.git?path=/Assets/PurrNet#release`

Do not bump the editor or PurrNet package without a heads-up and a matching
manifest / lock-file update.
